What is React?

React

(React.js or ReactJS) is a front-end JavaScript library for building user interfaces based on UI components which make the code easier to debug.

was developed by Meta Platforms, Inc.
(formerly Facebook).

is famous for building single-page applications,
mobile or server-rendered applications.

follows the MVC (Model View Controller) architecture.

follows the declarative programming paradigm.

suggests developers design views for each state of
an application and updates and renders components when data changes.

React history

React

was created by Jordan Walke, a software engineer at Facebook.

early prototype was called "FaxJS".

was first deployed on Facebook's News Feed in 2011 and later on Instagram in 2012.

was open-sourced at JSConf US in May 2013.

current version is 18.1.2

    Advantages and disadvantages

    React advantages:

    • React creates its own virtual DOM.
    • ReactJS helps to create awesome UI.
    • Extremely easy to start.
    • Component and Data patterns improve readability which helps to maintain larger apps.
    • React can be used with other frameworks and bundlers.
    • React simplifies the complete process of the scripting environment.
    • It facilitates advanced maintenance and increases productivity.
    • Ensures faster rendering.
    • Provision of a script for mobile app development.
    • React JS comes with a helpful developer toolset.
    • A lot of companies use React.

    React disadvantages:

    • High pace of development.
    • Poor documentation.
    • View Part.
    • JSX as a barrier.
